== Introduction
The Meyer template was chosen for this Software Requirements Specification
document. While this document follows the provided template for the most part, the
following changes were made:

* Added this Introduction section.
* Removed author biography section. This section was deemed unnecessary.
* Removed individual section control info. These were removed because the
  granularity was not appropriate for the changes and rounds of feedback used for
  this document. The overall document control info is still included, but it
  was updated to include the "Developers" and "Change" columns.
* Added an S7.adoc under the system directory. This addition corresponds to the
  request made in the Hazard Analysis document to include the additional 
  safety and security requirements made there into the SRS.
