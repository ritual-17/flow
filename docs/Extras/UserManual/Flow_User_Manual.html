1. What is Flow?

Flow is a keyboard-driven diagramming application. You create shapes, connect them with lines, and add rich text labels — all without leaving the keyboard.

Flow is inspired by Vim. Like Vim, it uses modes: the action a key performs depends on the mode you are currently in. Once you learn the modes, you can build complex diagrams quickly and precisely.

Flow renders text labels using Typst, a markup language that supports maths, symbols, and styled text. Completed diagrams export to PDF.

Note The status bar at the bottom of the window always shows your current mode and the next available actions. Check it whenever you are unsure what to do.

2. Your First Diagram — Quick Start

Follow these steps to build a simple two-shape diagram connected by an arrow. This covers the primary workflow.

Step 1 — Place two shapes

  1. Open Flow. You start in Normal mode.
  2. Press i to enter Insert mode.
  3. Press r, then click the canvas to place a rectangle.
  4. Press r again and click a different location to place a second rectangle.
  5. Press Esc to return to Normal mode.

Step 2 — Label the first shape

  1. Use h / j / k / l to move the cursor over the first rectangle.
  2. Press t to open the label editor.
  3. Press i inside the editor to start typing, then type your label (for example: Start).
  4. Press Esc twice — once to leave insert mode in the editor, and again to compile the label and close the editor.
Tip Repeat Step 2 for the second shape and label it End.

Step 3 — Connect the shapes with an arrow

  1. Move the cursor near the first rectangle in Normal mode.
  2. Press A (capital A) to enter Anchor-Line mode. Anchor dots appear on nearby shapes.
  3. Use h / j / k / l to navigate to the anchor dot on the right side of the first rectangle.
  4. Press Space to attach the line to that anchor.
  5. Move the cursor toward the second rectangle. Press e to add an arrowhead at the end.
  6. Navigate to the anchor dot on the left side of the second rectangle and press Space to finish the line.
  7. Press Esc to return to Normal mode.
Important When you move a shape that has an anchored line attached, the line updates automatically to follow the shape.

Step 4 — Save

  1. Press Ctrl+S to save your diagram.

3. Understanding Modes

Every key in Flow behaves differently depending on the active mode. Press Esc at any time to return to Normal mode and cancel the current action.

ModeHow to enterWhat you can do
NormalEsc from any modeMove the cursor, navigate shapes, issue commands
Inserti in NormalPlace new shapes on the canvas
Visualv in NormalSelect shapes to move, copy, or delete
Linea in NormalDraw a free multi-point line
Anchor-LineA in NormalDraw a line anchored to shape connection points
Textt near a shapeEdit the shape's label in the Typst editor
Auto-Link InsertI in NormalPlace a new shape and auto-connect it to the previous shape
Note The mode name in the status bar changes every time you switch modes. If a key does not behave as expected, check the status bar to confirm your current mode.

5. Place and Move Shapes

Place a shape

  1. Press i to enter Insert mode.
  2. Press the key for the shape you want.
  3. Click on the canvas where you want to place it. Flow centres the shape on the click point and returns to Insert mode.
  4. Press Esc when you are done placing shapes.
KeyShape
rRectangle
sSquare
cCircle
tText box

Move a shape

  1. Press v to enter Visual mode.
  2. Move the cursor over the shape and press Space to select it.
  3. Use Shift+h/j/k/l to nudge the shape (10 px per press).
  4. Use Ctrl+Shift+h/j/k/l for larger moves (50 px per press).
  5. Press Esc to deselect and return to Normal mode.

Delete a shape

  1. Press v to enter Visual mode.
  2. Move the cursor over the shape and press Space to select it.
  3. Press d to delete the selection.
Warning Deleting a shape also removes all lines connected to it. Press u immediately to undo if this was not intended.

6. Add Text to a Shape

Every shape can carry a label. Labels are written in Typst markup and rendered as an image on the canvas. You do not need to know Typst to write plain text — type normally and Flow renders it as-is.

Edit a label

  1. Move the cursor near the shape in Normal mode.
  2. Press t. A text editor opens on the nearest shape.
  3. Press i to start typing.
  4. Type your text.
  5. Press Esc to stop typing, then press Esc again to compile and close the editor.
Note The label editor uses Vim key bindings. Press i before you begin typing — you start in the editor's Normal mode, not Insert mode.

Typst markup — common formatting

Markup you typeResult on canvas
*bold text*Bold text
_italic text_Italic text
$ x^2 + y^2 $Inline maths: x² + y²
#sym.arrow.rRight arrow symbol →
(blank line between paragraphs)New paragraph
Tip If you only need plain text with no special formatting, simply type your text. No markup is required.

7. Draw Lines

A line in Flow is a polyline: a series of straight segments connected at vertices. You place each vertex with Space.

Draw a free line

  1. Press a in Normal mode to enter Line mode.
  2. Move the cursor to where you want the first vertex.
  3. Press Space to place the vertex.
  4. Move the cursor and press Space again to add more vertices.
  5. Press Esc to finish the line and return to Normal mode.

Add arrowheads

While drawing — at any point before pressing Esc — press e to cycle through arrowhead states:

StateDescription
No arrowheadsPlain line (default)
End arrowheadArrow at the last point
Both arrowheadsArrows at both ends
Start arrowheadArrow at the first point only

Press e again to cycle back to no arrowheads.

Line vertex dots

After you finish a line, each vertex appears as a small semi-transparent dot. These dots are snap points — you can connect other lines to them in Anchor-Line mode (see Section 8).

8. Connect Shapes with Anchored Lines

An anchored line is attached to a specific connection point (anchor) on a shape. When you move the shape, the line follows it automatically.

Anchor points

Each shape displays its anchor points as small dots when you are in Anchor-Line mode. Use h / j / k / l to navigate between them.

Draw an anchored line between two shapes

  1. In Normal mode, press A to enter Anchor-Line mode.
  2. Move the cursor near the source shape. Its anchor dots appear.
  3. Use h / j / k / l to move to the anchor where you want the line to start.
  4. Press Space to attach the line start to that anchor.
  5. Move the cursor toward the target shape. Its anchor dots appear.
  6. Navigate to the desired anchor on the target shape.
  7. Press Space to attach the line end and complete the connection.
  8. Press Esc to return to Normal mode.
Tip Press e at any point while drawing to cycle arrowhead states on the line.

Connect a line to a vertex on another line

You can also snap to vertices on existing lines:

  1. Enter Anchor-Line mode (A).
  2. Move the cursor near the target line. Its vertex dots become snap targets.
  3. Navigate to the desired vertex and press Space to attach.

Auto-Link Insert mode

Auto-Link Insert mode places a new shape and connects it to the previous shape automatically:

  1. In Normal mode, press I (capital I) to enter Auto-Link Insert mode.
  2. Press a shape key (r, s, c, or t).
  3. Click on the canvas. Flow places the shape and draws a connecting line from the previous shape.
  4. Each subsequent shape you place connects to the one before it, forming a chain.
  5. Press Esc to finish.

9. Select, Copy, and Delete

Select shapes

  1. Press v in Normal mode to enter Visual mode.
  2. Move the cursor over a shape and press Space to toggle its selection.
  3. To select all shapes at once, press Ctrl+A.
  4. Press Esc to deselect and return to Normal mode.

10. Save and Export

Save your diagram

KeysAction
Ctrl+SSave the current file

Export to PDF

  1. Press Ctrl+E to export to PDF.
  2. Choose a destination in the save dialog.

Flow renders all shapes and labels — including Typst maths and symbols — into the PDF.

Important All labels must be compiled (you must have pressed Esc to close the editor at least once) before you export. Uncompiled labels do not appear in the PDF.

Import a PDF slide

  1. In Normal mode, press f to import a PDF file as a slide placeholder on the canvas.
